#  Delivery App

A React-based Food Delivery Web Application built using Vite and Bootstrap.

##  Features
- Menu Page with food items
- Cart Page UI (static design)
- Contact Form
- Order Method section
- Responsive UI using Bootstrap

## Tech Stack
- React JS
- Vite
- Bootstrap

##  Project Purpose
This project is created for internship practice and frontend development learning.
